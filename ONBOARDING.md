## Summary: Where "Prefer not to say" Applies

| Step | Question / Field | How "Prefer not to say" works |
|------|-------------------|-------------------------------|
| 10 | Ethnicity | Option in list (value: `prefer_not_to_say`) |
| 11 | Marital status | UI-only option; stored in `profile_completion_prefer_not` |
| 16 | Religion | Option in list + UI card; stored in list or prefer_not |
| 17 | Political views | UI-only option; stored in `profile_completion_prefer_not` |
| 20 | Smoking | Per-field UI option; stored in `profile_completion_prefer_not` |
| 20 | Drinking | Per-field UI option; stored in `profile_completion_prefer_not` |
| 20 | Marijuana | Per-field UI option; stored in `profile_completion_prefer_not` |
| 21 | Has children | UI-only option; stored in `profile_completion_prefer_not` |
| 22 | Wants children | UI-only option; stored in `profile_completion_prefer_not` |

All other optional steps can be **skipped** (left empty); no separate "Prefer not to say" option.  
Required steps (1–5) have no skip and no "Prefer not to say." Step 6 (verification selfie) is optional and skippable only.
